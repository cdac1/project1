Browser JSGuard


Thank you for using Browser JSGuard.

History of changes

Version 3.x.x

Fixed the vulnerability in alerting mechanism.

Version 2.x.x

Due to new add-on packaging, minimal version supported is now Firefox 21.

Version 1.0.0

This version works from Firefox 13 onwards.

Limitation

Per window private browsing mode on Firefox 20 is not compatible with SDK panels feature. This feature is used by this extension for context menu.
So, on Firefox 20, Browser JSGuard context menu(Reporting) is deactivated.
It is strongly recommended to upgrade to Firefox 21 and above to have a fully operational Browser JSGuard.
